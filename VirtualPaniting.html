Virtual Painting System

Design for kids' drawings as a lightweight digital canvas, with many kids' psychology-based colour theories fonts and simple and elegant UI designs. Leapy canvas is not only a simple canvas like "paint" application but also a canvas which can see their drawing scores in real-time. Kids have two main features called "Free Hand" & "Color Book"; with these options able to draw with a free mindset and draw with leapy suggestions with the score management system. The most important part of the system is this Leap motion sensor, which provides a free hand drawing feature without hanging anything with their hands. Furthermore, the Artwork saving system is also a natural time-saving system with recent artworks, which can save recent artworks with given scores and other prediction scores. The kid and parent easily understand the child's drawing and painting progress.


System Overview

This component works as an individual part with a leap motion sensor controller; this sensor controller can track and handle hand gestures via the leapy application. That sensor tracks and inputs all hand gestures to an interface, and, Interface identifies what is going on with their fingers and the currently used gestures. In addition, our UI also works with sensor input; that is why we implemented this as a score-based freehand drawing application. It can draw and paint anything with a free mindset, and the application also provides an additional feature called “Color book”, which provides drawing suggestions to kids to draw on the canvas. As input, we mainly get it from huge canvases that a child draws. Line drawing, shape drawing, colour use and some numbers and tests must be analysed and saved to a database within a few minutes. Moreover, the leap motion tracking sensor allows tracking details via image processing and object identification technologies.

In addition to that, with this leap motion tracking visualizer software we can achieve and track our hands moments accurately and see how its work with our application.

Finally, identified what type of elements must consider when evaluating a drawing of a children’s

2. Accuracy of drawing
3. Color differences and blending options
4. Child interactivity with canvas
5. Button accuracy
6. Bugs and swapping errors while moving with pages

Hardware requirements

Although this is a simple and minimalist project, when it comes to hardware parts need to assemble or purchase a few things for use in this compact light white application.

• A LED or LCD Computer based smart TV (Size is matter)


Figure Computer connected Smart TV


An intelligent android TV is a better choice for this “leapy” application because when it comes to drawing, smart TV is getting bigger and bigger, kids easily view drawing screens widely, and giant screens are more interactive for all ages. Workflows and parents also can see the drawing and painting without going to kids near the area.

• Internet Connection

Figure Fiber Internet Connection


Most parents use a fibre optic internet connection for their internet solution, but some still use a 4G LTE Connection. However, it does not always matter. Leapy products always work with both connections. However, we all know a Fibre connection is faster than a 4G connection, same as that theory; if a parent uses a fibre connection for an application, then the application generates scores faster than a 4G connection generating speed. With Fibre, leapy works well, and as product developers, we recommend using a fibre connection that is better than a 4G internet connection for better results.

• Leap motion sensor or graphic Tablet

Figure Leap Motion Sensor Controller


The Leap Motion Controller is a small device that connects with a PC or Mac and enables users to manipulate digital objects with hand motions. 1 Working with other hardware, the Leap Motion controller adds a new way to interact with the digital world. 2 Programs designed to interpret gesture-based computing allow the user to play games, create designs, and learn in a ‘hands-on’ way.

Figure Graphic Tablet


A graphics tablet is an input device the user can use to draw images on a computer screen without using a mouse or keyboard. It consists of a flat, touch-sensitive pad and a drawing device, usually a pen or pen type. Also known as a drawing tablet or drawing pad, this tablet is ideal for artists and anyone who wants to manipulate an on-screen cursor with the natural feel of a pen-like object. If someone can afford a leap Motion Controller, they can move to a graphics tablet at a low cost. Both devices can quickly get in touch with the leapy drawing application, and the whole system works simultaneously.
